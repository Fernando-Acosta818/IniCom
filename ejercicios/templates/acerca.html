Acerca de este proyecto

Este sitio web cumple con el objetivo de unificar los proyectos de los cursos de Iniciación al Cómputo, siendo el proyecto de Iniciación al Cómputo II la aplicación web que alberga en una base de datos los proyectos de Iniciación al Cómputo I

Las herramientas utilizadas en este proyecto conjunto son las siguientes:

  • Backend: Framework web Django usando el lenguaje de programación Python
  • Frontend: Bootsrap 5.3
  • Base de datos: SQLite3
  • Los ejercicios de Iniciación al Cómputo I fueron realizados con Javascript
